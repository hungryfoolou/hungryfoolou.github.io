Import csv format of mysql's data into navicat

发表于 2019-04-11 | 更新于 2019-04-20 | 分类于 web crawler

目标

想把师兄从mysql导出的cnnvd数据导入navicat中。

问题描述

目前有数据的xls格式，用navicat导入xls格式数据时navicat停止了工作

解决方法

  1. 先用excel打开xls格式的数据文件，另存为csv格式的数据文件；
  2. 用notepad++打开csv格式的文件，选择菜单的编码->转换为utf-8格式；
  3. 在navicat里新建数据库cnnvd，数据库名为cnnvd，字符集为utf8 -- UTF-8 Unicode，排序规则为utf8_general_ci；
  4. 在导入向导的第2步选择编码格式为utf-8格式；
  5. 在导入向导的第6步，把类型从默认的varchar改为longtext，因为有的属性的值的长度比较大。

感想

  1. 为了避免导入数据的格式错误，需保证将要导入的数据和建立的数据库的编码方式相同。
  2. 数据一旦变得较大，就容易出现问题，额，希望问题少些。
  3. 师兄也给了xml格式的数据文件，没有尝试导入，应该只要编码格式一致就可以导入。
