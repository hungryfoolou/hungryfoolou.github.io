将大文件的xls格式转换为csv格式

发表于 2019-08-16 | 分类于 Python

Problem Description

用excel将大文件的xls格式转换为csv格式时，excel崩溃了

Solution

有2种解决办法

1. 增大电脑的虚拟内存，详见参考，具体步骤如下：

  1. 进入win10系统桌面，鼠标右键“我的电脑”，选择“属性”打开
  2. 在新出现的界面中选择“高级系统设置”打开。
  3. 在弹出的界面中点击“高级”，在性能的右下边，选择“设置”打开。
  4. 在“性能选项”界面中切换到“高级”，选择“更改”打开。
  5. 将虚拟内存界面中的勾去除，选择“自定义”，托管的系统选择C盘。
  6. 设置虚拟内存建议最多不超过实际内存的2倍，设置完毕后点击确定，重启系统即可应用设置。
  7. 虚拟内存设置大小一般应该为物理内存的1.5倍为好，如果物理内存为2G，虚拟内存应该设置为3G=3072MB，如果物理内存为4G，那虚拟内存应该设置为6G=6144MB。我把内存设置为了8G（最小值）-10G（最大值）

易上手的字幕制作软件ArcTime

发表于 2019-05-05 | 分类于 Entertainment

ArcTime简介

由于英语课程跨文化交际要做一个小组作业，我负责找视频，然而下载的字幕有些问题，正好自己也想学习一下字幕编辑的相关知识，在知乎上了解到ArcTime这个易用的软件。你看使用教程就会发现这个软件对新手是真的友好。

编辑视频完整的步骤

我使用ArcTime编辑视频字幕一般会通过下面几个步骤：

  1. 在youtube寻找视频
  2. 在clipconverter下载寻找到的视频
  3. 在downsub下载字幕
  4. 修改字幕
    （1）修改srt字幕文件后缀为txt；
    （2）用ArcTime打开txt，点击右中部的铅笔按钮来修改。或者打开Sublime text，按ctrl+h，点击左下角的”Aa”左边的”Regular expression”，在”Find What”中输入<([^<>]*)>，在”Replce With”中输入一个空格符，然后再通过类似的操作把>=2个的换行符替换为一个换行符，>=2个的空格替换为一个空格，即”\n{2,}”替换为”\n”，把” {2,}”替换为” “（{2,}前面有一个空格，而且不包括双引号）
    （3）每一行文字对应一个字幕块，过短的行可以和上一行或下一行合并为同一行。过长的行修改为两行或者可以先暂时不管，后来在ArcTime中通过插入新轨道（点击软件左下角的加号），这样就可以显示在视频中一次显示多行文字；
    （4）修改错别字和错误的标点符号；
    （5）寻找英文字母的第一个单词，首字母改为大写；
    （6）把每行英文字母的第一个单词前的空格去掉。
  5. 看第一个教程，导入视频和字幕后，交替按JK键（先按J键按住不放，然后本句字幕要结束时，马上松开J键，按K键）把字幕给添加到视频中，如果要在视频中添加某些英文单词的中文释义，需要点ArcTime左下角的加号，添加新轨道，点击右下角的”快速拖拽创建工具/JK键拍打工具”在轨道上画一段字幕块（或者点击双击新轨道按回车生成一段字幕块），拖动字幕块到合适的位置后，双击字幕块，修改字幕块内容，按回车键保存内容，右击该字幕块，设定样式，分配一个新样式，修改后按ctrl+s保存（可看该教程）。最后点击菜单栏的“导出”的“快速压制视频”从而把字幕添加到视频中并导出视频（导出视频的设置看该教程）
